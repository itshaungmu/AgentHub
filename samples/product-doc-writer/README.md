# Product Documentation Writer

**AgentHub 官方样板 Agent**

一个产品文档撰写助手，帮助创建清晰、专业、易于理解的产品文档。

## 特性

- 📝 **文档撰写**: 编写功能文档、用户指南、API 文档
- ✨ **文档优化**: 改进可读性和完整性
- 📋 **格式规范**: 统一的文档风格
- 🌐 **中英双语**: 支持中英文文档

## 安装

```bash
agenthub install product-doc-writer --target-workspace ./workspace
```

## 使用示例

```
用户: 帮我写一个用户登录功能的文档

助手: # 用户登录

## 简介
用户登录功能允许用户通过邮箱和密码访问其账户。

## 快速开始

1. 打开应用
2. 点击"登录"按钮
3. 输入邮箱和密码
4. 点击"提交"

## 详细说明

### 登录方式
- 邮箱 + 密码
- 手机号 + 验证码
- 第三方账号 (Google/GitHub)

### 安全特性
- 密码加密存储
- 登录失败限制
- 两步验证支持

## 常见问题

**Q: 忘记密码怎么办？**
A: 点击"忘记密码"，通过邮箱重置。
```

## 包含内容

- 📝 AGENTS.md - Agent 定义和行为规范
- 🎭 SOUL.md - Agent 性格和写作风格
- 👤 USER.md - 用户画像
- 🆔 IDENTITY.md - 身份信息
- 🔧 TOOLS.md - 可用工具说明
